## 📋 Descripción

**Fraseesencial** es una plataforma educativa interactiva diseñada para ayudar a estudiantes y profesionales a dominar la estructura esencial del español: **QUIÉN + VERBO + QUÉ + A QUIÉN**. A través de ejercicios prácticos, flashcards y seguimiento de progreso, los usuarios aprenden a distinguir entre elementos esenciales y accidentales en la construcción de textos.

### 🎯 Características Principales

- ✅ **100+ ejercicios interactivos** con diferentes niveles de dificultad
- ✅ **Sistema de flashcards** para práctica progresiva
- ✅ **Creador de ejemplos** personalizados
- ✅ **Ejercicios de ordenación** de elementos gramaticales
- ✅ **Redacción guiada** de párrafos (narrativos, descriptivos, expositivos, argumentativos)
- ✅ **Seguimiento detallado** de progreso y estadísticas
- ✅ **Interfaz responsive** con Bootstrap 5
- ✅ **Autenticación** de usuarios
